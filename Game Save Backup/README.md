# Game Save Backup

Программа для управления конфигурациями резервного копирования игровых сохранений. Позволяет пользователю создавать профили для игр, указывать путь к exe-файлу, папке с сейвами и папке для бэкапов, запускать игру через клиент или напрямую, а после закрытия игры автоматически выполнять резервное копирование сохранений в указанное место

### Описание программы

**Модуль списка конфигураций**
  - Отображает все созданные конфигурации в алфавитном порядке
  - Хранит индекс конфигураций в `configs_index.json`
  - Поддерживает выбор, запуск и редактирование существующих конфигураций
  - Автоматически определяет, какая конфигурация активна по умолчанию, исходя из последнего запуска

**Модуль создания и редактирования конфигурации**
  - Позволяет создать профиль игры с полями: название игры, URL запуска через клиент, путь к exe, путь к сейвам и путь для бэкапов
  - Выполняет валидацию обязательных полей и проверку на дубликат названия
  - При редактировании обновляет существующий JSON-файл конфигурации без добавления новой записи в индекс

**Модуль запуска игры**
  - Запускает выбранную или последнюю использованную конфигурацию
  - Показывает статус запуска, информацию о процессе, путях к сейвам и бэкапам, а затем отслеживает закрытие игры
  - После завершения игры инициирует процесс резервного копирования

**Модуль бэкапа сейвов**
  - Создает папку бэкапа по имени игры и подпапку с датой и временем
  - Копирует игровые сейвы в новую папку
  - Архивирует предыдущие бэкапы и хранит не более 10 архивов, удаляя самый старый при превышении лимита
  - Отображает обратный таймер закрытия информационного окна после завершения бэкапа

**Модуль интерфейса пользователя**
  - Состоит из основного окна списка конфигураций и окна создания/редактирования конфигурации
  - Содержит кнопки выбора файлов и папок, подсказку по получению URL из игрового клиента и статусы выполнения операций
  - Управляет состояниями элементов интерфейса: активен/заблокирован, выделение, подсветка ошибок, отображение сообщений
