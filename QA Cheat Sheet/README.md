# QA Cheat Sheet

Программа чит-листов, позволяющая конвертировать знания о проверках в переиспользуемые шаблоны и быстро собирать из них документацию тестирования

### Описание программы

Вместо того чтобы каждый раз заново вспоминать, что нужно проверить, тестировщик один раз формирует [чит-лист](https://github.com/Eventoi/SDLC-Basis/blob/feature/Testing/Theory/Documentation%20Preparation.md#%D0%B8%D1%81%D0%BF%D0%BE%D0%BB%D1%8C%D0%B7%D0%BE%D0%B2%D0%B0%D0%BD%D0%B8%D0%B5-%D1%87%D0%B8%D1%82-%D0%BB%D0%B8%D1%81%D1%82%D0%B0) и затем многократно переиспользует его в разных проектах и сценариях

Программа помогает:
- сократить время на подготовку тестов
- повысить покрытие за счёт готового списка проверок
